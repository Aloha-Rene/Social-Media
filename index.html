SOCIAL MEDIA

View Project View Code

This Social Media page was inspired by FaceBook. The page embodies the essence of a FaceBook page design, and has many avenues for Social interaction. The members viewing the site immediately sees a familiar social media platform. When scrolling down the page, the next section features a comment area .

This project was made with HTML, CSS, and Sass. Gulp was used for compiling Sass. The page is fully responsive for all sizes of screen, from ultrawide monitors to extra small phones. Click on either the image or "View Project" button above to see the live project, or click the "View Code" button above to view the code on GitHub.
